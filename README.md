# Samsung ScoreFlow Glasses Pro

![Samsung ScoreFlow Glasses Pro](assets/scoreflow-presentation.jpeg)

Landing page conceptual para presentar **Samsung ScoreFlow Glasses Pro**, unas gafas de realidad aumentada pensadas para musicos. El producto proyecta partituras en el campo de vision, avanza paginas automaticamente mientras el musico interpreta y ofrece herramientas de estudio asistidas por IA.

> Proyecto web academico y de portfolio. El producto es ficticio y se utiliza como ejercicio de diseno, desarrollo frontend y comunicacion de producto.

## Demo

Abre `index.html` en el navegador para ver la pagina principal.

```text
index.html
```

La ficha de producto y reserva esta disponible en:

```text
detail.html
```

## Que incluye

- **Landing page de producto** con narrativa comercial, hero visual, video, secciones de valor y preguntas frecuentes.
- **Pagina de detalle y reserva** con precio, selector de color, llamadas a la accion y resumen de caracteristicas.
- **Selector interactivo de acabados** para cambiar entre Negro Grafito, Plata Titanio y Azul Zafiro.
- **Carrusel automatico** con pausa al pasar el cursor, controles manuales y barra de progreso.
- **Videos integrados** para presentar el producto y mostrar una simulacion de uso.
- **Metronomo funcional** en la pagina de detalle, creado con JavaScript y Web Audio API.
- **Cuenta atras de lanzamiento** hasta el 5 de enero de 2027.
- **Diseno responsive** preparado para escritorio y dispositivos moviles.

## Idea del producto

ScoreFlow Glasses Pro esta disenado para resolver un problema real de musicos, estudiantes y docentes: mantener la concentracion durante la interpretacion sin tener que apartar las manos o la mirada para pasar paginas.

La experiencia combina:

- Partitura en realidad aumentada.
- Reconocimiento de audio en tiempo real.
- Paso de pagina automatico.
- Biblioteca musical sincronizada.
- Modo concierto sin distracciones.
- Asistente de practica con IA para detectar errores y proponer ejercicios.

## Tecnologias utilizadas

- **HTML5** para la estructura semantica de las paginas.
- **CSS3** para layout responsive, animaciones, estados visuales y presentacion de producto.
- **JavaScript vanilla** para la interaccion de carruseles, selectores, video, metronomo y cuenta atras.
- **Web Audio API** para generar los pulsos del metronomo.
- **Assets multimedia propios del proyecto**: imagenes, SVG y videos.
